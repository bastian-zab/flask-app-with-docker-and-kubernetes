DEPLOY DOCKER CONTAINER TO KUBERNETES CLUSTER

Docker is an open source containerization tool that packages applications into small units called containers. A docker container imageis a standalone software package that contains all the requirements for an application to run. Without docker, a developer can send a running code to a tester but upon running, the code might not run run on the tester's system. However, with Docker, an app is packaged with all the requirents and dependencies hence a Docker container can run on any platform.
